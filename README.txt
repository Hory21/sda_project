Fisierul executabil este "main.executable", si acesta ruleaza sub sistemul de
operare Linux Ubuntu (probabil ruleaza sub orice sistem de operare UNIX-based,
dar nu garantez asta).
Pentru recompilare se foloseste comanda "./compile.sh" cu conditia ca in
directorul proiectului sa nu fie decat fisierele lui.
Fisier de intrare: "data.in"
Fisier de iesire: "data.out"
